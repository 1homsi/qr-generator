# QR Generator

A stylish QR code generator built with React. Generate customizable QR codes for URLs, vCards, WiFi networks, Email, SMS, and plain text.

**Live:** https://1homsi.github.io/qr-generator/

## Features

- **Multiple QR types** — URL, Text, vCard, Email, SMS, WiFi
- **Full customization** — dot styles, corner styles, colors, gradients
- **Logo overlay** — embed an image in the center of the QR code
- **Download** — export as PNG

## Tech Stack

- [React 19](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [qr-code-styling](https://github.com/kozakdenys/qr-code-styling)

## Getting Started

```bash
pnpm install
pnpm dev
```

## Deployment

Automatically deployed to GitHub Pages on every push to `main` via GitHub Actions.
